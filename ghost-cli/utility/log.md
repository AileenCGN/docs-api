This command shows you the log of your Ghost site (not of the CLI!). The command requires you to navigate into your Ghost installation folder.

### Arguments

```
ghost log name
```

+ run `ghost ls` and copy the name of your site

### Options

```
--follow, -f
```

+ follow the log line
