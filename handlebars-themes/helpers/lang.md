Usage: `{{lang}}`

### Description

`{{lang}}` is a helper to output the configured site language.

### Example Code

Read the [translate helper page](/api/handlebars-themes/helpers/translate/) for further information:

```handlebars:title=default.hbs
<html lang="{{lang}}">
...
</html>
```
